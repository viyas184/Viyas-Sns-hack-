# LoanIQ: One-Stop Financial & Trading Hub ⚡

A comprehensive, responsive financial ecosystem designed to act as a singular endpoint for loan underwriting, credit score estimation, and actionable wealth generation.

## 🚀 Key Features

* **Global Currency Support**: Native `Intl.NumberFormat` integration switching seamlessly between USD, EUR, GBP, INR, JPY, and more!
* **AI Loan Underwriting**: Predictive models utilizing Explainable AI to transparently generate loan approvals based on credit history and debt.
* **CIBIL Score Estimator**: Real-time Javascript simulator generating local credit scores (Range: 300-900) based on on-time payment ratio and utilization metrics.
* **50/30/20 Budgeting Engine**: Instantaneous wealth planner dividing User Input Income cleanly into optimal Saving/Spending/Needs structures.
* **Smart Financial Chatbot**: Context-aware simulated assistant integrated natively to provide tips regarding Mutual Funds, SIPs, Intraday Trading, and Credit Building.

## 💻 Tech Stack
- **Frontend**: Vanilla Javascript, HTML5, CSS3 Glassmorphism UI
- **Backend API**: Node.js, Express.js (Includes Python FastApi mock for environment compatibility)

## 🛠️ How to Run Locally

### 1. Start the Backend API (Port 8000)
```bash
cd backend
node server.js
```
*Note: Make sure port 8000 is open. The API processes underwriting requests and trading tips.*

### 2. Start the Frontend Application (Port 3000)
```bash
cd frontend
node serve.js
```
*Navigate to `http://localhost:3000` in your web browser to enjoy the dynamic dashboard.*

## 🏆 Hackathon Scope
Built specifically for financial analysis & stock market integration. The goal of this application is to completely replace scattered financial structures with one centralized tool where users can evaluate lines of credit while simultaneously learning to multiply their capital safely.
